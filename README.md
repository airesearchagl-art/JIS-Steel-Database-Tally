鉄骨数量集計システム (JIS-Steel-Database-Tally)
JIS規格に基づいた鋼材の重量および表面積を、ブラウザ上で素早く集計できる実務向け積算補助ツールです。

🚀 特徴
豊富なJISデータベース: H形鋼、コラム、パイプ、山形鋼、溝形鋼、FB、PLなどの主要規格を網羅。

精密なデータ選択: 丸パイプやCチャンなどの軽量部材は、肉厚違い（1.6t〜4.5t等）まで細かく選択可能。

グループ管理機能: 階層ごとに部材を分けて集計し、グループごとの合計値も自動算出。

データポータビリティ:

プロジェクト保存: 作業内容をJSON形式で保存し、後日再読み込み可能。

CSV出力: 集計結果をエクセル等で加工できるようCSV出力に対応。

完全ローカル動作: サーバー不要のHTML/JavaScript構成。ブラウザさえあれば環境を問わず利用可能。

🛠 使い方
index.html をブラウザで開きます。

「新規グループ追加」でセクション（例：1F、階段部、屋根など）を作成。

部材の大分類・中分類・小分類を順に選択。

長さ（m）と本数を入力すると、重量と表面積がリアルタイムで計算されます。

必要に応じて「プロジェクト保存」や「CSV出力」を行ってください。

📦 収録鋼材リスト
重量鋼材: H形鋼（広・中・細幅）、角形鋼管（BCR/BCP）、円形鋼管（STK/STKN）

形鋼: 等辺/不等辺山形鋼、溝形鋼（[）、T形鋼（CT）

軽量・二次部材: 角/丸パイプ（各肉厚対応）、リップ溝形鋼（Cチャン）

板・その他: 平鋼（FB：3mm〜25mm厚各種）、丸鋼（RB）、鋼板（PL）
