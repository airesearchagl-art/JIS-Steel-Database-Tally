# 鉄骨数量集計システム (JIS-Steel-Database-Tally)

JIS規格に基づいた鋼材の重量および表面積を、ブラウザ上で素早く集計できる実務向け積算補助ツールです。インストール不要で、現場や事務所のブラウザからすぐに利用可能です。

## 🚀 主な機能

- **膨大なJIS規格データベース**: H形鋼、コラム、山形鋼、溝形鋼、T形鋼、鋼管、FB、PLなどの主要なJIS規格を網羅。
- **精密な部材選択**: 丸パイプ（一般・手すり）やCチャンなど、肉厚（t）のバリエーションも詳細に選択可能。
- **グループ管理システム**: 部位別（例：1F、階段、屋根）に行をグループ化し、個別の小計と総合計を算出。
- **データポータビリティ**: 
  - **プロジェクト保存**: 入力データをJSON形式でローカルに保存し、いつでも再編集が可能。
  - **CSV出力**: エクセルでの加工や報告書作成に便利なCSV形式での書き出しに対応。
- **完全ローカル動作**: 外部サーバーにデータを送信しないため、セキュリティ面でも安心して実務に使用できます。

## 🛠 使い方

1. `index.html` をブラウザ（Chrome, Edge等）で開きます。
2. **「新規グループ追加」**をクリックして、集計セクションを作成します。
3. ドロップダウンメニューから、**大分類 > 中分類 > 小分類 > サイズ** の順に選択します。
4. **長さ(m)** と **本数** を入力すると、重量(kg)と表面積(㎡)が自動算出されます。
5. 作業を中断する場合は **「プロジェクト保存」** を、結果をエクセルで使いたい場合は **「CSV出力」** を利用してください。

## 📦 収録部材データ

- **重量鋼材**: H形鋼（広・中・細幅）、角形コラム（BCR/BCP）、円形鋼管（STK/STKN）
- **形鋼**: 等辺山形鋼、不等辺山形鋼、不等辺不等厚山形鋼、溝形鋼（[）、T形鋼（CT）
- **軽量・二次部材**: 角/丸パイプ（肉厚バリエーション網羅）、リップ溝形鋼（Cチャン：1.6t〜4.5t）
- **板材・その他**: 平鋼（FB：3mm〜25mm厚の多サイズ展開）、丸鋼（RB）、鋼板（PL）

## 📜 開発の歩み (Development History)

本ツールは、積算業務の「手入力によるミス」と「JIS表を参照する手間」を削減するために開発されました。

1. **Phase 1: 基本機能の実装**
   - ブラウザベースの計算ロジックと、H形鋼・鋼管の基本JISデータの搭載。
2. **Phase 2: 形鋼データの拡充**
   - 実務で多用される山形鋼（アングル）や溝形鋼（チャンネル）の全サイズを統合。不等辺・不等厚などの特殊規格にも対応。
3. **Phase 3: 二次部材・軽量材の精密化**
   - 「丸パイプの肉厚違い」や「Cチャンの1.6t/2.3t/3.2t/4.5t」といった、詳細な規格を網羅。平鋼(FB)のサイズバリエーションを大幅に拡張。
4. **Phase 4: プロジェクト管理機能の統合**
   - 複数部位の同時集計を可能にするグループ機能、およびデータの保存・読込機能を実装し、現在の「鉄骨数量集計システム」として完成。
